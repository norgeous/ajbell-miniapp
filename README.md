# ajbell-miniapp

## Install

```sh
git clone https://github.com/norgeous/ajbell-miniapp
cd ajbell-miniapp
npm install
```

## How to use

| Command | What it does |
|--|--|
| `npm run dev` | Runs Nextjs in development mode |
| `npm run storybook` | Runs Storybook in development mode |
